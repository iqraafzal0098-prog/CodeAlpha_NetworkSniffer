# CodeAlpha Network Sniffer

A Python-based network packet sniffer built using Scapy to capture and analyze real-time network traffic. Developed as **Task 1** of the CodeAlpha Cyber Security Internship.

## 📌 Features
- Captures live network packets on Windows
- Analyzes packet headers (source/destination IP, protocol type)
- Displays TCP, UDP, and ICMP packet details
- Built with Scapy and Npcap

## 🛠️ Technologies Used
- Python 3
- Scapy library
- Npcap (packet capture driver for Windows)
